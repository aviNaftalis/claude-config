# claude-config

My [Claude Code](https://claude.com/claude-code) setup — global `CLAUDE.md`,
`settings.json`, and a colored statusline with a caveman-mode badge.

![demo](demo.gif)

## Install

```sh
git clone https://github.com/aviNaftalis/claude-config.git
cd claude-config
./install.sh
```

Restart Claude Code.

## Dependencies

- `jq`
- `git` (optional — enables the branch segment)
- A Nerd Font terminal (for the branch glyph)

Plugins referenced in `settings.json`:
[caveman](https://github.com/JuliusBrussee/caveman),
[claudebin](https://github.com/wunderlabs-dev/claudebin).
